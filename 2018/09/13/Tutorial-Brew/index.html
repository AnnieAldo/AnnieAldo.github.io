brew cask

安装

根据官网提供的方法安装
即直接在cmd输入：
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

随后安装几个基础包：
brew install wget git

基础命令

brew search [pkg] 查找
brew list 查看目前所有安装的包
brew update 更新所有包
brew upgrade [pkg] 更新某个包
brew cleanup ([pkg]) 删除旧版本的安装包
brew outdated 查看哪些已安装的包需要更新

cask

brew cask install [app] 安装
brew cask uninstall [app] 卸载
brew cask list 查看目前所有安装的app
brew cask cleanup 删除所有安装包

./brew_cask_upgrade.sh 使用脚本升级所有，也可以直接把.sh文件拖到命令行窗口

如果执行这一步的时候被报错”Permission denied”
可以使用 chmod 777 brew_cask_upgrade.sh 修改使用权限

自己下载好需要安装的包

brew是单线程下载，比较慢。

  1. 使用brew cask install [app]进入下载界面，会看到一个下载地址，看到后停止当前下载
  2. brew --cache查看brew下载目录，进入目录
  3. 删除刚才下载中的文件，替换为在下载地址下载的文件。注意，根据下载中的文件的文件名 重命名 下载的文件
  4. 重新brew cask install [app]
  5. 一般下载的文件比较大，brew cask cleanup清一下安装包

推荐应用

brew cask install qlcolorcode qlimagesize qlmarkdown qlstephen quicklook-csv quicklook-json
ql或者quicklook开头的app全都用于预览，使预览文件的时候能够格式化文件

brew cask install shiftit
shiftit是窗口分配app，和Moom相似

brew cask install atom
